- **Database**: PostgreSQL with Drizzle ORM
- **UI Components**: Shadcn/ui, Radix UI, Lucide icons
- **Deployment**: Vercel-ready with serverless configuration

## Quick Deploy to Vercel

1. **Database Setup**: Create free PostgreSQL at [neon.tech](https://neon.tech)
2. **Deploy**: Import this repository to Vercel
3. **Environment Variables**:
   ```
   DATABASE_URL=your_postgresql_connection_string
   NODE_ENV=production
   VITE_API_URL=https://your-app.vercel.app
   ```
4. **Initialize Database**: Run `npm run db:push`
